# 高速铁路桥上无砟轨道几何形位调整课件

本仓库用于发布“高速铁路桥上无砟轨道几何形位调整课件”的网页版本。

## 在线访问

部署到 GitHub Pages 后，直接访问仓库的 Pages 地址即可打开课件。

## 本地运行

双击 `index.html` 可在浏览器中打开课件。建议使用最新版 Chrome 或 Microsoft Edge。

请保持 `index.html`、`styles.css`、`app.js`、`model-viewer.js` 和 `assets` 文件夹在同一目录下，否则样式或三维模型可能无法正常加载。

## 文件说明

- `index.html`：课件入口页面
- `app.js`：课件交互逻辑
- `styles.css`：页面样式
- `model-viewer.js`：三维模型展示脚本
- `assets/`：三维模型资源
- `高速铁路桥上无砟轨道几何形位调整课件.zip`：原始最终成果压缩包
